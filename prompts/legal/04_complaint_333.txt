3-3-3 FRAMEWORK EXPLAINED:
═══════════════════════════════════════════════════════

**3 COUNTS (Legal Claims):**
Each count must be a separate legal theory (e.g., Fraud, Breach of Contract, Negligence)

**3 CLAIMS PER COUNT:**
Each count should have 3 supporting sub-theories or alternative bases

**3+ ELEMENTS PER CLAIM:**
For each claim, explicitly plead all required elements

EXAMPLE STRUCTURE:

COUNT I: FRAUD
  Claim 1A: Intentional Misrepresentation
    ✓ Element 1: False representation of material fact
    ✓ Element 2: Knowledge of falsity (scienter)
    ✓ Element 3: Intent to induce reliance
    ✓ Element 4: Justifiable reliance
    ✓ Element 5: Resulting damages
    
  Claim 1B: Fraudulent Concealment
    ✓ Element 1: Concealment of material fact
    ✓ Element 2: Duty to disclose
    ✓ Element 3: Intent to defraud
    ✓ Element 4: Reliance on concealment

DRAFTING STANDARDS & TIPS:
═══════════════════════════════════════════════════════

1. FRAUD SPECIFICITY (FRCP 9(b)):
   - WHO made the statement
   - WHAT was said (exact words if possible)
   - WHEN it was said (date and time)
   - WHERE it was said (location)
   - HOW it was communicated (oral, written, email)
   - WHY it was false
   - HOW plaintiff relied
   - WHAT damages resulted

2. PLEADING STANDARDS:
   - Use FRCP/State Rule standards
   - Plead facts, not conclusions
   - Be specific and concrete
   - Number all paragraphs
   - Use short, declarative sentences

3. INCORPORATION BY REFERENCE:
   - Incorporate prior paragraphs at start of each count
   - Maintains all allegations throughout

4. EXHIBITS:
   - Attach key documents
   - Reference as "Exhibit A," "Exhibit B," etc.
   - Incorporate by reference in body

5. DAMAGES:
   - Plead "according to proof" for flexibility
   - Also plead minimum amount if known
   - Specify types: actual, consequential, punitive
   - Include statutory damages if applicable

6. JURY DEMAND:
   - Always demand jury unless strategic reason not to
   - Demand on "all issues so triable"

Remember: A good complaint tells a story, pleads law precisely, and sets up the entire case for success.
