## üö® TROUBLESHOOTING

**Problem:** "Permission denied" when running scripts
**Solution:**
```bash
chmod +x script_name.sh
./script_name.sh
```

**Problem:** "Python module not found"
**Solution:**
```bash
pip3 install module_name
# Or: pip install module_name
```

**Problem:** Can't find files
**Solution:**
```bash
# In a-Shell:
pwd  # See current directory
ls  # List files
cd ~  # Go to home
cd Private-Claude  # Go to project
```

**Problem:** Git authentication fails
**Solution:**
```
1. GitHub Settings ‚Üí Developer Settings
2. Personal Access Tokens ‚Üí Generate new token
3. Use token as password
```
