## ğŸ› TROUBLESHOOTING

### "My Zap isn't running"

**Check:**
1. Is the Zap turned ON? (blue toggle in Zapier)
2. Is it published? (check top right in editor)
3. Are all apps connected? (My Apps section)
4. Check Zapier History for errors

**Fix:**
- Turn Zap OFF then ON again
- Disconnect and reconnect apps
- Delete and recreate the Zap

### "Emails not arriving"

**Check:**
1. Spam folder
2. Email address is correct (terobinsonwy@gmail.com)
3. Gmail is connected in Zapier
4. Test the Zap manually

**Fix:**
- Reconnect Gmail account
- Add zapier@zapier.com to contacts
- Check Gmail filters

### "Google Sheets not updating"

**Check:**
1. Sheet name matches exactly
2. Headers match field names
3. Google Sheets is connected
4. Sheet is not deleted

**Fix:**
- Reconnect Google Sheets
- Verify sheet permissions
- Create fresh sheet

### "Webhook URL not working"

**Check:**
1. URL is copied correctly (no spaces)
2. Zap with webhook is ON
3. Webhook trigger is published
4. No firewall blocking

**Fix:**
- Create new webhook trigger
- Get fresh URL
- Test with verify_zapier.html
